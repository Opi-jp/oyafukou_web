## 開発用スクリプトについて

開発・デバッグ用のスクリプトは `.gitignore` で除外されます：
- `test-*.js` - テスト用スクリプト
- `check-*.js` - 確認用スクリプト
- `debug-*.js` - デバッグ用スクリプト

これらのパターンに従って一時的なスクリプトを作成することで、リポジトリをクリーンに保てます。
